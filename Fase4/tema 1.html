Etiquetas HTML

Una etiqueta HTML es un markup language, lo que significa que está escrito con códigos que puede
una persona sin que sea necesario compilarlo primero. En otras palabras, el texto en una página web
está «marcado» con estos códigos para dar instrucciones al navegador web sobre cómo mostrar el texto.
Estas etiquetas de marcado son las propias etiquetas HTML.

Definición de etiquetas HTML,
recuperado de https://www.iebschool.com/blog/que-es-etiqueta-html-analitica-usabilidad/

Formato de textos

  • "b" Aplica negrita al texto, incluido entre las etiquetas, es equivalente a "Strong"
  • "i" Aplica cursiva al texto, includio entre las etiquetas, es equivalente a "em"
  • "u" Aplica subryado
  • "font" Delimita un texto con un formato de fuente determinado definido por sus propiedades:
    "font size", "font color", "font face"
  • "h1, h2, h3, h4, h5, h6" Indican 6 niveles de formato de encabezados, en los que
    "h1" delimitaría el tipo de fuente de mayor tamaño.
Celaya Luna, A. (2014). Creación de páginas web: HTML 5 (pág. 1 a 60).
Recuperado de https://elibro-net.bibliotecavirtual.unad.edu.co/es/ereader/unad/56045

Formato de párrafos

  • "br" Introduce un salto de línea
  • "p" Delimita un párrafo de texto
  • "hr" Inserta una línea horizontal
  • "div" Se utiliza para delimitar una sección dentro del documento, de forma que agrupe
    un número de elementos para luego añadirle un estilo determinado o realizar
    operaciones sobre ese bloque específico.
Celaya Luna, A. (2014). Creación de páginas web: HTML 5 (pág. 1 a 60).
Recuperado de https://elibro-net.bibliotecavirtual.unad.edu.co/es/ereader/unad/56045

Creación de listas

Lista no numerada

  • "ul" Inicio de la lista
    • "li" Primer elemento de la lista
    • "li" Segundo elemento de la lista
    • "li" Tercer elemento de la lista
  • "ul" Cierre de la lista
    • Lista numerada

    • "ol" Inicio de la lista
      1. "li" Primer elemento de la lista
      2. "li" Segundo elemento de la lista
    • "ol" Cierre de la lista
    Celaya Luna, A. (2014). Creación de páginas web: HTML 5 (pág. 1 a 60).
    Recuperado de https://elibro-net.bibliotecavirtual.unad.edu.co/es/ereader/unad/56045

    Tablas

    • "Table" Define dónde comienza y termina la tabla
    • "tr" Indica el comienzo y el fin de cada una de las filas de la tabla
    • "td" indica el comienzo y el fin de cada una de las columnas o celdas dentro de las filas.
    Celaya Luna, A. (2014). Creación de páginas web: HTML 5 (pág. 1 a 60).
    Recuperado de https://elibro-net.bibliotecavirtual.unad.edu.co/es/ereader/unad/56045

    Formularios

    • "form" Indican el comienzo y el fin del formulario
    • "label" Delimitan una etiqueta de texto dentro del formulario
    • "input" Indican el principio y final de algún control de entrada de datos por parte
      del usuario. Pueden ser de varios tipos, según se defina su atributo type
    • "Select" Delimita el principio y el fin de una lista desplegable de opciones.
      cada opción se indica mediante la etiqueta "option".
    • "textarea" Permite introducir un texto largo.
    Celaya Luna, A. (2014). Creación de páginas web: HTML 5 (pág. 1 a 60).
    Recuperado de https://elibro-net.bibliotecavirtual.unad.edu.co/es/ereader/unad/56045
