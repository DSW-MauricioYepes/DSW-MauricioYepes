Sitio web para el aprendizaje de las Etiquetas básicas de HTML, Atributos y valores

Este sitio tiene como propósito, fomentar el aprendizaje en las temáticas correspondientes
a las etiquetas básicas de HTML, atributos y valores con la finalidad de dinamizar, y diseñar
contenidos multimediales e interactivos.

Etiquetas HTML

Una etiqueta HTML es un markup language, lo que significa que está escrito con códigos que puede
una persona sin que sea necesario compilarlo primero. En otras palabras, el texto en una página web
está «marcado» con estos códigos para dar instrucciones al navegador web sobre cómo mostrar el texto.
Estas etiquetas de marcado son las propias etiquetas HTML.

Definición de etiquetas HTML,
recuperado de https://www.iebschool.com/blog/que-es-etiqueta-html-analitica-usabilidad/

Atributos

Los atributos, son palabras especiales utilizadas dentro de la etiqueta de apertura,
para controlar el comportamiento del elemento. Los atributos de HTML son un modificador de un tipo
de elemento de HTML. Un atributo tampoco modifica la funcionalidad por defecto de un tipo de elemento
o proporciona funcionalidad a ciertos tipos de elementos incapaces de funcionar correctamente sin ellos.
En sintaxis HTML, un atributo se añade a una etiqueta de inicio de HTML.

Definición de Atributos HTML, recuperado de https://es.wikipedia.org/wiki/Atributo_HTML

Descripción

Los atributos de HTML generalmente se muestran como una pareja nombre-valor, separados por =,
y están escritos dentro de la etiqueta de inicio de un elemento, después del nombre del elemento.

Definición de Atributos HTML, recuperado de https://es.wikipedia.org/wiki/Atributo_HTML

Valores

Son las propiedades que se le pueden asignar a los elementos.

Definición sobre valores HTML, recuperado de https://es.wikibooks.org/wiki/Lenguaje_HTML/El_c%C3%B3digo

Descripción

Por ejemplo, a html se le puede asignar un atributo para indicar en qué idioma está escrito
el documento (lang) y podemos asignarle como valor un código de idioma ("es" para el español,
"en" para el ingles...) (html lang="es")...(/html). O sea: (elemento atributo="valor")...(/elemento). Por supuesto un elemento puede tener varios atributos: (elemento atributo="valor" atributo="valor")...(/elemento).

Definición sobre valores HTML, recuperado de https://es.wikibooks.org/wiki/Lenguaje_HTML/El_c%C3%B3digo
