Sitio web para el aprendizaje de las Etiquetas básicas de HTML, Atributos y valores

Este sitio tiene como propósito, fomentar el aprendizaje en las temáticas correspondientes
a las etiquetas básicas de HTML, atributos y valores con la finalidad de dinamizar, y diseñar
contenidos multimediales e interactivos.

Etiquetas HTML

Una etiqueta HTML es un markup language, lo que significa que está escrito con códigos que puede
una persona sin que sea necesario compilarlo primero. En otras palabras, el texto en una página web
está «marcado» con estos códigos para dar instrucciones al navegador web sobre cómo mostrar el texto.
Estas etiquetas de marcado son las propias etiquetas HTML.

Definición de etiquetas HTML,
recuperado de https://www.iebschool.com/blog/que-es-etiqueta-html-analitica-usabilidad/

Formato de textos

  • "b" Aplica negrita al texto, incluido entre las etiquetas, es equivalente a "Strong"
  • "i" Aplica cursiva al texto, includio entre las etiquetas, es equivalente a "em"
  • "u" Aplica subryado
  • "font" Delimita un texto con un formato de fuente determinado definido por sus propiedades:
    "font size", "font color", "font face"
  • "h1, h2, h3, h4, h5, h6" Indican 6 niveles de formato de encabezados, en los que
    "h1" delimitaría el tipo de fuente de mayor tamaño.
Celaya Luna, A. (2014). Creación de páginas web: HTML 5 (pág. 1 a 60).
Recuperado de https://elibro-net.bibliotecavirtual.unad.edu.co/es/ereader/unad/56045

Formato de párrafos

  • "br" Introduce un salto de línea
  • "p" Delimita un párrafo de texto
  • "hr" Inserta una línea horizontal
  • "div" Se utiliza para delimitar una sección dentro del documento, de forma que agrupe
    un número de elementos para luego añadirle un estilo determinado o realizar
    operaciones sobre ese bloque específico.
Celaya Luna, A. (2014). Creación de páginas web: HTML 5 (pág. 1 a 60).
Recuperado de https://elibro-net.bibliotecavirtual.unad.edu.co/es/ereader/unad/56045

Creación de listas

Lista no numerada

  • "ul" Inicio de la lista
    • "li" Primer elemento de la lista
    • "li" Segundo elemento de la lista
    • "li" Tercer elemento de la lista
  • "ul" Cierre de la lista
    • Lista numerada

    • "ol" Inicio de la lista
      1. "li" Primer elemento de la lista
      2. "li" Segundo elemento de la lista
    • "ol" Cierre de la lista
    Celaya Luna, A. (2014). Creación de páginas web: HTML 5 (pág. 1 a 60).
    Recuperado de https://elibro-net.bibliotecavirtual.unad.edu.co/es/ereader/unad/56045

    Tablas

    • "Table" Define dónde comienza y termina la tabla
    • "tr" Indica el comienzo y el fin de cada una de las filas de la tabla
    • "td" indica el comienzo y el fin de cada una de las columnas o celdas dentro de las filas.
    Celaya Luna, A. (2014). Creación de páginas web: HTML 5 (pág. 1 a 60).
    Recuperado de https://elibro-net.bibliotecavirtual.unad.edu.co/es/ereader/unad/56045

    Formularios

    • "form" Indican el comienzo y el fin del formulario
    • "label" Delimitan una etiqueta de texto dentro del formulario
    • "input" Indican el principio y final de algún control de entrada de datos por parte
      del usuario. Pueden ser de varios tipos, según se defina su atributo type
    • "Select" Delimita el principio y el fin de una lista desplegable de opciones.
      cada opción se indica mediante la etiqueta "option".
    • "textarea" Permite introducir un texto largo.
    Celaya Luna, A. (2014). Creación de páginas web: HTML 5 (pág. 1 a 60).
    Recuperado de https://elibro-net.bibliotecavirtual.unad.edu.co/es/ereader/unad/56045

    Atributos

    Los atributos, son palabras especiales utilizadas dentro de la etiqueta de apertura,
    para controlar el comportamiento del elemento. Los atributos de HTML son un modificador de un tipo
    de elemento de HTML. Un atributo tampoco modifica la funcionalidad por defecto de un tipo de elemento
    o proporciona funcionalidad a ciertos tipos de elementos incapaces de funcionar correctamente sin ellos.
    En sintaxis HTML, un atributo se añade a una etiqueta de inicio de HTML.

    Definición de Atributos HTML, recuperado de https://es.wikipedia.org/wiki/Atributo_HTML

    Descripción

    Los atributos de HTML generalmente se muestran como una pareja nombre-valor, separados por =,
    y están escritos dentro de la etiqueta de inicio de un elemento, después del nombre del elemento.

    Definición de Atributos HTML, recuperado de https://es.wikipedia.org/wiki/Atributo_HTML

    Atributos comunes

    La mayoría de elementos pueden tomar cualquiera de varios de los atributos comunes:

    • El atributo "id" (identidad) proporciona un identificador único en el ámbito de todo
      el documento para un elemento.
    • El atributo "class" proporciona una manera de clasificar elementos similares.
    • El atributo "style" Un autor puede utilizar propiedades no presentacionales de código
      atribucional de "style" para un elemento particular.El atributo de estilo se puede
      utilizar en cualquier elemento de HTML.
    • El atributo "title" (título) se suele usar para unir explicación subtextual a un elemento.
      En la mayoría de los navegadores, este atributo se muestra como lo que a menudo se llama
      un tooltip o globo.
    Definición de Atributos HTML, recuperado de https://es.wikipedia.org/wiki/Atributo_HTML

    Valores

    Son las propiedades que se le pueden asignar a los elementos.

    Definición sobre valores HTML, recuperado de https://es.wikibooks.org/wiki/Lenguaje_HTML/El_c%C3%B3digo

    Descripción

    Por ejemplo, a html se le puede asignar un atributo para indicar en qué idioma está escrito
    el documento (lang) y podemos asignarle como valor un código de idioma ("es" para el español,
    "en" para el ingles...) (html lang="es")...(/html). O sea: (elemento atributo="valor")...(/elemento). Por supuesto un elemento puede tener varios atributos: (elemento atributo="valor" atributo="valor")...(/elemento).

    Definición sobre valores HTML, recuperado de https://es.wikibooks.org/wiki/Lenguaje_HTML/El_c%C3%B3digo

    Ejemplos

    Imágenes

    • img src="dirección de la imagen" Inserta una imagen que se encuentra en la ruta indicada
      "por dirección de la imagen". También se pueden incluir una serie de propiedades.
    Creación de enlaces
    • (a href="http://www.ejemplo.com") Nombre del enlace (/a) Cuando se pulsa sobre el texto
      "Nombre del enlace" en la web, se va al vinculo indicado en la dirección de href.
    • (a href="mailto:ejemplo@ejemplo.com") Nombre del enlace (/a) Cuando se pulsa sobre el texto
      "Nombre del enlace" en la web, se abre una ventana para enviar un correo electrónico
      a la dirección indicada.
    Celaya Luna, A. (2014). Creación de páginas web: HTML 5 (pág. 1 a 60).

    Comentarios finales

    • Los atributos deben estar en la etiqueta inicial, nunca en la final.
    • El orden de los atributos dentro de la etiqueta es indiferente.
    • Deben separarse con un espacio en blanco del nombre del elemento y de otros atributos.
    • Atributo y valor deben estar unidos por el signo: =
    • Los valores deben estar encerrados con comillas, dobles o simples.
      Esto no es totalmente cierto pero actúa como si lo fuera.
    Definición sobre valores HTML, recuperado de https://es.wikibooks.org/wiki/Lenguaje_HTML/El_c%C3%B3digo
